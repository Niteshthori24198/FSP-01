Welcome to My Application

Project-Documentation : I have segregate my work into different parts and then links them all to make a fully function Full Stack application.

Front-End : It contains a user friendly GUI. Where user can register himself as well as can login into systym using his/her crendentials.

Backend : It contains the backend code from where user is accessing and storing information.

Modules in Front End

  • Home.html : It contains all front end GUI of Home Page.
  • About.html : It contains all details regarding Application.
  • Login/Register.html : It contains all front end GUI of User login and registration.
  • Style.css : It contains all the styling part for front end.
  • Index.js : It contains all the controlling code for front end and using that i am interacting with my backend.

Module in BackEnd

  • Routes : It contains all the routes where we are making http request for CRUD operations.
  • Model : It contains Database Structure and Connection of database with MongoDB Atlas.
  • Middleware : It contains all the Validator middlewares, Authentication and Logger Middlewares.
  • Server : It connects all the MVC part and make a connection beetween Front end and Backend.

External Modules

  • Express : It's used to make API request to our server.
  • Mongoose : It is used for making a connection beetween MongoDB Atlas and our Application.
  • Cors : It enables the cross origin connection so we can make request from our Front end to Backend.
  • Morgan : It's used for storing the information regarding the users Activities.
  • DotEnv : It's used for making important information Confidential in our Application.
